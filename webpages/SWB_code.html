Compiled SWB Executables and Source Code


Executables

SWB executables are currently available for Windows (32-bit and 64-bit) systems and Linux (64-bit) systems. There is no installation procedure needed; unzip or untar the archive files and ensure that the executables are placed into your path and are accessible from the command line.

Version 1.0.1 – Includes NETCDF support module

  • Windows (32-bit): http://wi.water.usgs.gov/Soil_Water_Balance/downloads/swb_windows_x86.zip, 1.2 MB, compiled September 10, 2012
  • Windows (64-bit): http://wi.water.usgs.gov/Soil_Water_Balance/downloads/swb_windows_x64.zip, 1.7 MB, compiled November 20, 2012
  • Linux (64-bit): http://wi.water.usgs.gov/Soil_Water_Balance/downloads/swb_ubuntu_x64.tar.gz, 6 MB

Test Case

A simple test case demonstrating the application of the Soil-Water-Balance Model may be downloaded here: http://wi.water.usgs.gov/Soil_Water_Balance/Example/swb_simple_example.zip , 400 KB

Source Code

A ZIP file that contains all of the source code for SWB may be downloaded here: http://wi.water.usgs.gov/Soil_Water_Balance/downloads/swb.zip 32.2MB

Source Code Documentation

Documentation for the current version of the source code may be viewed here: http://wi.water.usgs.gov/Soil_Water_Balance/SWB_code_docs/

Source Code Repository

Git is software that helps manage collaborative development of software projects. SWB now has a publicly accessible Git repository. The Git repository for current, developmental, and experimental versions of SWB may be found here [exit USGS]: https://github.com/smwesten-usgs/swb Keep in mind that the versions of SWB found on GitHub may contain undocumented features and incomplete additions to the code.
